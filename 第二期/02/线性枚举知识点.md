# 线性枚举 (Linear Enumeration)

## 概念
在线性枚举中，我们从头到尾依次扫描所有可能的选项，逐个判断是否满足条件，或者在过程中不断更新答案。  
它是一种**时间复杂度为 O(n)** 的暴力算法，是最基础也最常用的算法思想之一。

简单来说，线性枚举就是：
> “从第一个开始，一个一个试过去。”

这种方法在题目规模较小（如 `n ≤ 1e6`）时非常有效，  
同时，它也是许多复杂算法（如双指针、滑动窗口、动态规划）的基础。

---

## 基本的几种写法

### 1.基本模板
```cpp
for(int i=0;i<n;i++){
    //一个一个试过去
}
```


### 2.带条件判断
```cpp
for(int i=0;i<n;i++){
    if(...){
        //满足条件
    }
}
```

### 3.枚举并维护状态
线性枚举通常会与「计数器」技巧结合使用：
```cpp
int cnt=0;
for(int i=0;i<n;i++){
    cnt++;
    if(!condition) cnt=0;
    else if(cnt==3){
        //...
    }
}
```
> 第三段代码实现了啥功能? 思考一下?
<details>
<summary>答案</summary>
维护一个计数器cnt，记录当前连续满足条件的元素数量；<br>
如果当前元素不满足条件，则将计数器重置为 0；<br>
如果计数器达到 3，则执行相应操作。<br>
</details>

---

## 时间复杂度
线性枚举的时间复杂度一般为 **O(n)**，其中 n 为枚举的元素数量。

| 操作类型 | 时间复杂度 |
|-----------|-------------|
| 单次枚举 | O(1) |
| 遍历整个数组 | O(n) |

>如果在枚举过程中嵌套了别的过程，则复杂度可能上升到 O(n log n) 或 O(n²)。

## 例题

### Lc 1. 两数之和
题目链接：https://leetcode.cn/problems/two-sum/
#### 题目描述

给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 **和为目标值** `target`  的那 **两个** 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。

你可以按任意顺序返回答案。

#### 示例

```cpp
输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。
```
#### final solution
遍历第一个数，再遍历第二个数，如果两个数相加等于target，则返回两个数的下标。
时间复杂度为`O(n²)`,空间复杂度为`O(1)`。
```cpp
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        vector<int> res;
        for(int i=0;i<nums.size();i++){
            for(int j=i+1;j<nums.size();j++){
                if(nums[i]+nums[j]==target){
                    res.push_back(i);
                    res.push_back(j);
                    return res;
                }
            }
        }
        return res;
    }
};
```
